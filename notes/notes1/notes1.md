# Notes 1

## What is Markdown?
* A lightweight text formatting language
* Lets you write formatted text (headings, lists, links, code) using plain text symbols
* Commonly used for README files, documentation, and notes because it's easy and write

## What is Git?
* A version control system 
* Tracks changes to files over time so you can revert, compare, or collaborate safely
* Widely used by developers and IT teams to manage source code and configs

## What is GitHub?
* A cloud-based platform that hosts Git repositories
* Allows collaboration, backups, issue tracking, and project management
* Often used for sharing code, open-source projects, and school/work assignments

## What is Slack?
* A team communication platform
* Organizes conversations into channels 
* Commonly used in workplaces and schools instead of long email chains
