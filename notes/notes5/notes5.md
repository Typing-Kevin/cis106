## Commands for Navigating the File System

**1. `ls`**

**DEFINITION**
* The `ls` command is used to list files and directories in the current directory.
* You can also use `ls -a` to list any files and directories hidden in the current directory.

**USAGE/FORMULA**

`ls [options][directory]`

**EXAMPLES**
* Lists files in directory: `ls`
* Lists all files including hidden ones: `ls -a`
* List a specific directory: `ls Documents/`


**2. `pwd`**

**DEFINITION**
* `pwd` command displays the full path of the current working directory.

**USAGE/FORMULA**

`pwd`

**EXAMPLES**
* Show current directory: `pwd`


**3. `cd`**

**DEFINITION**
* `cd` command is used to change the current directory.

**USAGE/FORMULA**

`cd [directory path]`

**EXAMPLES**
* Move into a directory: `cd Documents`
* Go up one directory (parent directory): `cd ..`
* Go to home directory: `cd ~`
* Go to root directory: `cd /`


## Questions and Answers

**1. What is a variable?**
* A variable is a named value used to store data in memory that can be reused or changed.

**2. How do I use a variable?**
* You assign it with `=` and access it using `$`

**3. What is an environment variable?**
* An environment variable is a system-wide variable that affects how programs run.

**4. What is a user defined variable?**
* A user-defined variable is one you create yourself in the terminal or script.

**5. What is the root directory?**
* The root directory is the top-level directory in Linux.

**6. What does “Parent Directory” mean?**
* The parent directory is the folder one level above your current directory.

**7. What does “Current working directory” mean?**
* The current working directory is where you are right now in the filesystem.

**8. What is an absolute path? Include an example**
* An absolute path is the full path starting from root `/`.
Example: ```/home/kevin/Documents/file.txt```

**9.  What is a relative path? Include an example**
* A relative path is based on your current directory.
Example: ```/Documents/file.txt```

**10.  What is the difference between “Your home directory” and “The home directory”?**
* The home directory is the general folder where all users’ personal directories are stored.
* Your home directory is your specific personal folder inside it.
